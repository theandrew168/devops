# mariadb
Ansible role for installing [MariaDB](https://mariadb.org/)

## Configuration
| Variable | Default | Description |
| -------- | ------- | ----------- |
| `mariadb_root_password` | `~` | MariaDB root password |
| `mariadb_users` | `[]` | MariaDB users |
| `mariadb_databases` | `[]` | MariaDB databases |
